`x` is an S4 object of class <A>

> # S4
> compare(new("A", x = "1"), new("A", x = "1"))
v No differences

> compare(new("A", x = "1"), new("A", x = "2"))
`x@x`: "1"
`y@x`: "2"

> compare(new("A", x = "1"), new("B", x = "1"))
`is(x)`:     "A"
`is(y)`: "B" "A"
